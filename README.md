# Implementação de um cadastro de cliente de forma estruturada em Java

Este projeto apresenta a implementação de um **cadastro de clientes em Java**, desenvolvido utilizando o paradigma de **programação estruturada**.

O sistema foi construído com **apenas uma classe**, contendo diversos métodos responsáveis pelas operações de CRUD (Create, Read, Update e Delete).

##  Estrutura de Dados

Os dados dos clientes são armazenados utilizando **vetores (arrays)**, permitindo o cadastro de até **100 clientes**.

Cada cliente possui os seguintes atributos:
- **Id (int)**
- **Nome (String)**
- **CPF (String)**

Para isso, são utilizados **três vetores paralelos**, onde cada posição representa um cliente.

## Funcionalidades

O sistema disponibiliza as seguintes operações:

- **Incluir cliente**
- **Excluir cliente**
- **Alterar dados do cliente**
- **Consultar cliente** (por Id ou Nome)
- **Listar todos os clientes**

## Objetivo

Este projeto tem como objetivo demonstrar, de forma didática, o uso de:
- Vetores (arrays)
- Controle manual de dados
- Estruturas de repetição e decisão
- Organização de código em programação estruturada

---

Ideal para fins acadêmicos e para iniciantes que desejam compreender o funcionamento interno de um cadastro sem o uso de orientação a objetos avançada.
